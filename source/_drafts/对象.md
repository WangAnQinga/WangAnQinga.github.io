**所有对象都会从它的隐式原型上继承一个 constructor 属性**

**值得注意的是原型对象也拥有一个属性——constructor 指向其函数。**

```
var o = {};
o.constructor === Object; // true

var o = new Object;
o.constructor === Object; // true

var a = [];
a.constructor === Array; // true

var a = new Array;
a.constructor === Array // true

var n = new Number(3);
n.constructor === Number; // true
```
