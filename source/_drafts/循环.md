## 所有关于循环的

`for`、`while` 、 `do while`、`switch` 、`for in`、 `for of`、 `forEach`、 `map`、 `reduce`

## for

```
// 先判断后执行
for(var i=0;i<100;i++){
    console.log(i)
}

()中的i，相当于全局变量，除非用let
```


## while

// 先判断后执行

```
while (条件) 语句; 
 
//举例：
var i = 0; 
while (i < 100) { 
    console.log('i=' + i);
    i++; 
}
```

## do while

// 先执行，后判断     至少执行一次
```
do {
    语句
}while(条件)

let i = 0
do{
    console.log(i);
    i++
}where(i<100)
```

## map
map 是es5新增的循环方法
map 不会改变原数组，会返回一个新的数组

map ，让数组每个元素经过函数处理，并返回一个新数组
```
var arry = [1,2,3,4,5]
const newArry = arry.map(item => {
    item = item+1
    return item
})
console.log(newArry) [2,3,4,5,6]
```

## 总结：

while循环特点：先判断后执行；
do-while循环特点：先执行再判断，即使初始条件不成立，do-while循环至少执行一次，也就是说do-while循环比while循环多执行一次。
